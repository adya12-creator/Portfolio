# Adya Patnaik Portfolio

Modern personal portfolio built with React, Vite, and Tailwind CSS.

## Tech Stack
- React + Vite
- Tailwind CSS
- React Icons

## Run Locally
1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`
3. Production build:
   `npm run build`
4. Preview build:
   `npm run preview`

## Resume Setup
Place your resume file in `public/` with this exact name:
`Adya_Patnaik_FlowCV_Resume_2026-04-14.pdf`

## Customize Later
- Profile content: `src/data/portfolioData.js`
- Sections and layout: `src/sections/`
- Navigation and active scroll: `src/components/Navbar.jsx` + `src/App.jsx`
- Colors, typography, animation styles: `src/index.css`
